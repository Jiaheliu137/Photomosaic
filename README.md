# Photomosaic — Eagle 插件

将选中的图片转换为照片马赛克（Photomosaic），使用 Eagle 图库中的所有图片作为瓦片素材。

## 功能

- 选中一张或多张图片，一键生成照片马赛克
- 自动从 Eagle 图库索引所有可用图片作为瓦片
- 可调参数：输出宽度、精细度、瓦片比例、多样性、色彩还原度
- 两种颜色匹配模式：
  - **粗略**：使用 Eagle 内置调色板快速匹配，适合预览
  - **精确**：加载缩略图 5×5 子块匹配，效果更好，首次索引较慢
- 生成后可实时调节色彩还原度，无需重新生成
- 批量生成多张图片，生成过程中可自由切换查看，支持取消
- 每张图片可独立设置参数
- 画布支持滚轮缩放和拖拽平移
- 一键保存到 Eagle 图库
- 支持排除指定文件夹不参与瓦片索引
- 6 种 Eagle 主题自动适配

## 安装

1. 下载本插件文件夹
2. 打开 Eagle，进入 插件 → 导入本地插件
3. 选择 `Photomosaic` 文件夹即可

## 使用

1. 在 Eagle 中选中一张或多张图片
2. 打开 Photomosaic 插件
3. 调整参数后点击「生成当前」或「全部生成」
4. 生成完成后点击「保存到 Eagle」
