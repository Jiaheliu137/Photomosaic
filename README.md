# Photomosaic — Eagle 插件

将选中的图片转换为照片马赛克（Photomosaic），使用 Eagle 图库中的所有图片作为瓦片素材。

## 功能

- 选中一张或多张图片，一键生成照片马赛克
- 自动从 Eagle 图库索引所有可用图片作为瓦片
- 可调参数：输出宽度、精细度（每行瓦片数）、瓦片比例、多样性、色彩还原度
- 两种颜色匹配模式：
  - **粗略模式**：使用 Eagle 内置调色板，Lab 空间加权平均，KD-Tree 单色匹配，极快
  - **精确模式**：加载缩略图采样，5×5 子块 Lab 匹配，KD-Tree 粗筛 + 子块精排两阶段搜索，首次构建慢但匹配质量高
- L 通道加权：优先保证亮度匹配，远看轮廓更清晰
- 逐行渲染实时预览生成过程
- 生成后实时调节色彩还原度，无需重新生成
- 批量生成多张图片，生成过程中可自由切换查看，支持取消
- 每张图片独立参数，互不影响
- 画布支持滚轮缩放和拖拽平移，调参/生成不干扰视图状态
- 一键保存到 Eagle 图库「马赛克图片」文件夹，跳过已保存
- 瓦片索引按素材库+瓦片形状缓存到磁盘，带版本控制，换库自动清理
- 支持排除指定文件夹不参与瓦片索引
- 6 种 Eagle 主题自动适配

## 使用方式

1. 在 Eagle 中选中一张或多张图片
2. 打开 Photomosaic 插件
3. 调整参数后点击「生成当前」或「全部生成」
4. 生成完成后点击「保存到 Eagle」

## 目录结构

```
Photomosaic/          <- Eagle 插件目录
├── manifest.json
├── index.html
├── logo.png
├── css/
│   └── style.css
└── js/
    ├── utils.js      <- 工具函数、进度条、临时文件管理
    ├── color.js      <- RGB/Lab 转换、KD-Tree、子块距离、两阶段搜索
    ├── cache.js      <- 瓦片索引缓存、版本控制、素材库状态、文件夹排除
    ├── mosaic.js     <- 瓦片数据库构建（palette/sampled 5×5）、马赛克生成、叠色合成
    ├── viewer.js     <- 缩放平移查看器
    ├── ui.js         <- UI 状态管理、侧栏、参数面板、估算
    └── plugin.js     <- Eagle 生命周期、事件绑定、主题、窗口控制
```
